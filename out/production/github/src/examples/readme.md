### 案例总结

- 数据安全问题
  - 为什么出现问题?(这也是我判断多线程程序是否会有数据安全问题的标准)
    - 是否是多线程环境
    - 是否有共享数据
    - 是否有多条语句操作共享数据
  - 如何解决多线程安全问题?
    - 让程序没有安全问题出现的环境
      - 可以看出前两条肯定解决不了, 通过干掉第三条 处理问题
    - 同步代码锁
    ```
      // 注意这个obj 不同进程需要使用同一个对象
      synchronized (obj) {}
    ```
    - 注意这个东西可以在方法中声明
